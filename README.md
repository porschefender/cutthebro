## 它能帮你做什么？

录完很久的英文视频，素材又长又多，不知道从哪里剪起？

**CutTheBro 帮你：**

- 🎙️ 自动把视频/音频里的对话**转录成文字**（支持中英混合）
- 👥 自动识别**谁在说话**（多人对话也没问题）
- 🤖 用 AI **总结内容**，帮你找出最精彩的片段
- ✍️ 自动生成**抖音 / 小红书 / TikTok / Instagram 发布文案**
- ✂️ 你**点一下**想要的台词，程序自动帮你**剪出对应的音频**

| 类型 | 格式 |
|------|------|
| 🎬 视频 | `.mp4` `.mov` `.MP4` `.MOV` |
| 🎵 音频 | `.mp3` `.m4a` `.wav` `.aac` `.flac` |

> 💡 视频和音频都可以直接丢进去处理，不需要提前转换格式

## 安装教程

### 第一步：准备 HuggingFace 账号

1. 打开 👉 [huggingface.co](https://huggingface.co) 注册免费账号
2. 登录后点右上角头像 → **Settings** → **Access Tokens**
3. 点 **New Token**，名字随便写，权限选 **Read**，点 Create
4. 复制那串 `hf_` 开头的 token，**先存在备忘录里**

---

### 第二步：准备 Claude API Key

1. 打开 👉 [console.anthropic.com](https://console.anthropic.com) 注册账号
2. 点左边菜单 **API Keys** → **Create Key**
3. 复制那串 `sk-ant-` 开头的 key，**先存在备忘录里**
4. 点左边菜单 **Billing**，充值最低 $5（支持信用卡）

---

### 第三步：同意模型使用协议

用你的 HuggingFace 账号登录，依次打开下面三个链接，每个页面点击 **Agree**：

- 👉 [pyannote/speaker-diarization-3.1](https://huggingface.co/pyannote/speaker-diarization-3.1)
- 👉 [pyannote/segmentation-3.0](https://huggingface.co/pyannote/segmentation-3.0)
- 👉 [pyannote/speaker-diarization-community-1](https://huggingface.co/pyannote/speaker-diarization-community-1)

> ⚠️ 这一步不做的话，说话人识别功能会报错，一定要做！

---

### 第四步：一键安装 CutTheBro

打开 Mac 上的 **终端**（按 `Command + 空格`，搜索「终端」，回车）

把下面这行命令**复制粘贴**进去，按回车：

```bash
curl -fsSL https://raw.githubusercontent.com/porschefender/cutthebro/main/install.sh | bash
```

然后按照提示操作：
- 遇到需要输入密码的地方，输入你的 **Mac 开机密码**（输入时屏幕不显示字符，是正常的）
- 遇到问你 `y/n` 的地方，输入 `y` 回车
- 安装程序会提示你粘贴 HuggingFace Token 和 Claude API Key

整个过程大约 **15-30 分钟**，主要是下载 AI 模型比较慢，耐心等就好 ☕️

---

### 安装完成！

安装成功后，你会在 **Launchpad** 里看到 CutTheBro 的图标。

双击打开，浏览器会自动弹出操作界面 🎉

## 使用教程

### Tab 1：处理视频/音频 📥

```
粘贴文件夹路径 → 点「扫描」→ 勾选要处理的文件 → 点「开始处理」
```

1. 把你的素材放在一个文件夹里（比如 `Movies/节目素材`）
2. 在左边输入框粘贴这个文件夹的路径
   - 找路径的方法：右键点击文件夹 → 按住 `Option` 键 → 点「拷贝路径名」
3. 点「扫描」，程序会列出所有视频和音频文件
4. **绿色「已完成」**的文件是之前处理过的，不会重复处理
5. 勾选要处理的文件，点「开始处理」
6. 右边会实时显示进度和日志，耐心等待

> ⏱️ 处理速度参考：10分钟素材约需 5-10 分钟，1小时素材约需 30-50 分钟

---

### Tab 2：选取片段 ✂️

```
粘贴文件路径 → 自动加载分析报告 → 点选台词 → 导出音频
```

1. 在右上角粘贴对应文件的**完整路径**
2. 程序会自动匹配分析报告，显示带时间戳的中英对照文本
3. 点击你想要的台词行（英文或中文都可以点），选中变成高亮
4. 右边「已选片段」会显示你选的内容
5. 点「导出音频片段」，每一段单独保存为 MP3
6. 文件名自动以台词中文内容命名，保存在对应的项目文件夹里

## 输出文件在哪里？

所有文件保存在：

```
~/CutTheBro/outputs/20240101_Project/
```

文件夹名称是处理当天的日期，里面包含：

| 文件 | 说明 |
|------|------|
| `文件名.txt` | 完整转录文本 |
| `文件名_分析报告.txt` | AI 分析报告（含摘要、高光推荐、各平台文案） |
| `01_台词内容.mp3` | 你导出的音频片段 |

## 常见问题

**Q：处理一半卡住了怎么办？**
关掉 CutTheBro，重新打开，重新点「开始处理」，已完成的文件会自动跳过。

**Q：说话人识别不准确？**
嘈杂环境、多人同时说话时准确率会下降，这是正常的，手动调整一下就好。

**Q：AI 分析的文案不满意？**
分析报告是参考，你可以自由修改。把报告内容复制给 AI 助手让它帮你改写也很方便。

**Q：音频文件也可以直接处理吗？**
可以！支持 `.mp3` `.m4a` `.wav` `.aac` `.flac`，和视频文件一样操作。

**Q：怎么更新到新版本？**
在终端运行：
```bash
cd ~/CutTheBro && git pull
```
